# token-saver Documentation

## Description

This folder contains the documentation website source for token-saver.
The site is built with mdBook and published through GitHub Pages.

## Installation instructions

1. Install Rust (for project development): https://rustup.rs/
2. Install mdBook:
   - `cargo install mdbook`
3. Build docs locally from repository root:
   - `mdbook build docs`

## Usage examples

- Serve docs locally:
  - `mdbook serve docs --open`
- Build static docs:
  - `mdbook build docs`
