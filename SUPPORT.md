# Support

## Getting help

For usage questions, troubleshooting, and ideas:
- Open a GitHub Discussion (preferred)
- Open a GitHub issue for bugs or feature requests

## Before opening an issue

Please include:
- Your OS and shell
- The command you executed
- The summarized output and expected behavior
- If possible, a minimal reproducible example

## Security

For security issues, follow [SECURITY.md](SECURITY.md) and do not disclose
vulnerabilities publicly.
